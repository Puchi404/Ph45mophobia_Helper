# Ph45mophobia_Helper
Stoppwatch and Detection of a Ghost with the hunting speed


Anleitung zur Nutzung (Ph45m Stand 16.05.2026)
1. Stoppuhr
Start / Stopp: Tippe auf die große Stoppuhr-Karte.
Zeitfarben:
0:00 – 0:59→ Grün
1:00 – 1:30→ Rot („Dämon?“)
1:31 – 2:59→ Grün
ab 3:00→ Blau („Spirit?“)
Zurücksetzen: Klicke auf den „↺ Zurücksetzen“-Button.
2. Metronom
BPM einstellen:
Über den Schieberegler (Bereich passt sich automatisch an die gewählte Geschwindigkeit und Blutmond an).
Mit Tap Tempo (unten): Tippe 4× oder öfter auf den großen Button. Das Metronom übernimmt das Tempo. Mit „↺ Zurücksetzen“ startest du neu.
Start/Stopp: Klicke auf „▶ Start“ / „⏸    Stopp“.
Ton-Pause: Beim Bewegen des Reglers pausiert der Sound, um Störgeräusche zu vermeiden.
3. Geister-Voreinstellungen
Wähle einen Geist aus der Liste. Bei Geistern mit Geschwindigkeitsbereich (Hantu, Moroi, Thaye) wird automatisch der nächstliegende Zustand angezeigt.
Geschwindigkeit: Wähle 50 %, 75 %, 100 %, 125 % oder 150 %.
Blutmond: Aktiviere den Schalter, um den Blutmond-Faktor (×1,5) hinzuzufügen.
4. Filter („Alle Geister anzeigen“)
Ist die Checkbox deaktiviert, werden nur Geister angezeigt, die im Bereich von ±10 BPM um den aktuellen Metronomwert liegen.
Grün = naher Wert, Rot = exakter Treffer (±3 BPM).
Manuell ausgewählte Geister bleiben sichtbar.
5. Tap Tempo
Tippe im Rhythmus auf den „Tippe im Rhythmus“-Button. Nach 4 Taps stellt sich das Metronom automatisch ein.
Mit „↺ Zurücksetzen“ löschst du die Taps.
